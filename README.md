# SkillForge AI Driven Adadptive Learning and Exam Generator
SkillForge is an AI-based learning platform.

# Overview
It gives personalized learning content and automatically generates exams based on student performance.
The system adapts to the user's skill level and helps improve learning step by step.

# Features
Adaptive learning based on user performance.

AI-generated quizzes and exams.

Performance analysis and progress tracking.

Topic-wise question generation.

User-friendly dashboard.
